# PRANA-G Flutter UI (Mobile Only)

This folder contains a Flutter (Dart) UI-only rebuild of the current app.

## What is included
- Login screen
- Bottom tabs: Home, My Cattle, Alerts, Profile
- Health Check flow screen
- Matching color palette, gradients, card styling, and mock interactions

## What is not included
- Backend integration
- Camera/image API processing
- Real auth

## Run
1. Install Flutter SDK.
2. From this folder:
   - `flutter create .`
   - `flutter pub get`
   - `flutter run`
